SimpleWordScraper

Hello, this project was created as part my personal fun project.
To achieve the result the following tools / technologies are implemented:

  • Python Flask: Backend Server
  • Python Beautiful Soup: Webscraper
  • Chart.js: Data Visualization

Scrape Words

Scrape texts of all p-tags from the supplied URL and then counts words.

Scrape Sentences

Scrape texts of all p-tags from the supplied URL,
analyzes the sentiment of each sentence and then visualizes the result.
